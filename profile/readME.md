## 🏡 서비스 소개

> "내 집에서 가장 가까운 중고 거래, 동네.zip" <P> 동네.zip은 지역 밀착형 중고 거래 플랫폼으로, 판매자와 구매자 간 실시간 채팅을 통해 안전하게 직거래할 수 있는 서비스를 제공합니다.</p> <p> 프로젝트 종료 후, Redis 기반의 실시간 기능을 고도화하고, 코드의 안정성을 확보하는 리펙토링을 진행하였습니다 </p>

<br>

- 서비스명 : 동네.zip | 하이퍼로컬 중고거래 플랫폼
- 기간 : 2025.05.21~2025.07.05 (총 46일, 리펙토링)
